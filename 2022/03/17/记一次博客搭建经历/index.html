记一次博客搭建经历#标题

March 17, 2022 1510

仅做备份，简明搭建流程
win平台

需要用到的软件

  • github
  • hexo博客框架
  • zhaoo主题
  • git
  • node.js

    一般流程

github前期准备

github 注册，注册之后创建以**.github.io**结尾的Repository。

1. 填 mysoftlight.github.io
2. 选 Initialize this repository with a README
3. 点 create

装node.js

1. 选LTS版本  
2. 安装 一直点击下一步
3. 打开cmd 输 node -v  和 npm -v  若都显示版本号则成功

装git

1.  官网下32/64位程序解压即可

装hexo

1.  先创建一个名为blog的文件夹，然后 cd 切过去(或者直接在建好的文件夹下面右键选git bash打开)
2.  输入 npm instsll -g hexo-cli 之后用 hexo -v 验证是否成功

初始化 hexo

1.  hexo init myblog 文件名自定义 ，如果在自己设的文件夹下可直接使用 hexo init
2.  cd myblog 切入文件夹 如果在自己设的文件夹下可忽略
3.  npm install 安装依赖包
4.  hexo g 生成网页，展示的是自带网页
5.  hexo s 将生成的页面放在本地服务器
6.  浏览器输入 localhost:4000/ 就可以看到了
7.  git bash里按Ctrl+C停止服务器

Helloword 展示

1.  在博客路径下输入 hexo new "helloword"
2.  会在.\blog\source_posts路径下生成名为helloword.md的文件。可以随便写点什么然后保存关上。
3.  在git Bash 输 hexo g 然后输 hexo s
4.  刷新之前页面，会看到更新后的页面。

远程部署(一)

1.  密钥生成(Name和Email是github的用户名和邮箱) 
- git config --global user.name "Name"
- git config --global user.email "Email"
- ssh-keygen -t rsa -C "Email"
2.以上直接回车即可，保存默认位置密码空着就行。
3.  复制密钥 
- clip < ~/.ssh/id_rsa.pub

远程部署(二)

1.  github 头像->Settings->SSH and GPG kesys -> new SSH key
2.  名字随意 ，密钥放刚刚粘贴板里的
3.  测试连接
-  ssh -T git@github.com

修改blog配置

 deploy:
    type: git
    repo: git@github.com:mysoftlight/mysoftlight.github.io.git
    branch: master

写文章发布文章

1.  博客根目录 安装扩展
- npm i hexo-deployer-git
2.  git Bash下输入(G 生成静态页面 s启动本地服务器 d推送到github)
- hexo g
- hexo d 
3.  输入你的博客地址 https://你的用户名.github.io
